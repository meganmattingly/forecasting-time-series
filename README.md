# Forecasting and Time Series Analysis

This repostitory contains the weekly assignments and final project that were completed during the Forecasting and Time Series course (BANA 7050). The goal was to recieve an introduction to time series data and discuss several common forecasting techniques. Each assignment focuses on a different method, but they all use the same data set and build on each other. The final project is a complete time series analysis that includes EDA, decomposition, model development, and cross-validation to create the best final forecasts. 

Although this course was taught using R, I incorporated many of the same techniques into my [capstone research project](https://github.com/meganmattingly/capstone-research-project), which was done in Python. 

## Assignment Description

Assignment 2: Data Description, Exploratory Analysis, and Decomposition

- This script summarizes and visualizes time series data and creates baseline seasonal naive forecasts. 

Assignment 3: ARIMA Modeling

- This script tests for mean and variance stationarity, compares ARIMA orders, checks residuals for autocorrelation, and creates forecasts for the selected model. 

Assignment 4: Model Selection and Validation

- This script splits the time series into training and test data, sets up the cross-validation scheme, compares metrics different time horizons, and creates final forecasts for the better performing model. 

Assignment 5: Forecasting with Prophet

- This script introduces Meta's Prophet model, experiments with hyperparameter tuning, performs cross-validation, and creates final forecasts. 

Final Project

- This script synthesizes the concepts covered in each weekly assignment to create a comprehensive analysis of the time series.
